# 🚀 Hashuri

<img src="gopher.webp" align="right" width="250px"/>

**Hashuri** is a simple standard package, that contains a `web server` and `HTTP` Functions with some "Tea".

Hashuri is a web micro-framework written in Go because, i'm learning go and i just saw some packages in Go that have amazing features! So I thought, "Why don't I make MY OWN PACKAGE?", and here it is, made with love and lots of coffee!

## 📦 Instalation

Use this command:

```powershell
go get perfecttea.net/hashuri
```

And it's installed!
